# AI Chat

Tab Manager Pro includes a **conversational AI assistant** accessible from the Chrome side panel.

## Access the chat

Click the Tab Manager Pro icon in Chrome's Side Panel, or right-click the extension icon > **Open side panel**.

## Features

### Free conversation

Ask any question about your tabs:

- *"Which tabs are related to development?"*
- *"How many YouTube tabs do I have open?"*
- *"Summarize what I'm currently doing"*

The AI has access to the **list of your current tabs** (title + URL) to answer with context.

### Natural language commands

The chat understands **intents** and executes actions directly:

| Command | Examples |
|---|---|
| Close duplicates | *"Close duplicates"*, *"Remove duplicate tabs"* |
| Group tabs | *"Group my tabs"*, *"Organize the tabs"* |
| Sort tabs | *"Sort the tabs"*, *"Order by domain"* |
| Close inactive | *"Close inactive tabs"*, *"Close old tabs"* |
| Close a tab | *"Close the YouTube tab"*, *"Close Netflix"* |
| Save session | *"Save the session"*, *"Save session Project X"* |
| Restore session | *"Restore the session"*, *"Open the last session"* |
| Create workspace | *"Create a Dev workspace"* |
| Open workspace | *"Open the Dev workspace"* |
| Search for a tab | *"Search for the GitHub tab"* |

### Quick suggestions

The chat offers **suggestion buttons** for the most common actions:

- Close duplicates
- Group tabs
- Sort tabs
- Save session
- Close inactive

### History

Conversations are **persisted** between sessions. You can view the history or clear it.

::: info Bilingual support
Commands are understood in both **French** and **English**.
:::
