# Commands

Tab Manager Pro recognizes **natural language intents** from the AI chat. Simply type what you want to do.

## Command list

### Tab management

#### Close duplicates
Identifies tabs with the same URL and closes the copies.

```
"Close duplicates"
"Remove duplicates"
"Ferme les doublons"
"Supprime les doublons"
```

#### Sort tabs
Reorganizes tabs by domain name, in alphabetical order.

```
"Sort the tabs"
"Order the tabs"
"Trie les onglets"
```

#### Close inactive
Closes all non-active and non-pinned tabs.

```
"Close inactive tabs"
"Close old tabs"
"Ferme les onglets inactifs"
```

#### Close a specific tab
Closes tabs matching a keyword.

```
"Close the YouTube tab"
"Close Netflix"
"Ferme l'onglet YouTube"
```

#### Search for a tab
Find a tab by its title or URL.

```
"Search for tab React"
"Find the GitHub tab"
"Cherche l'onglet React"
```

#### Group tabs
Triggers AI grouping (equivalent to the "Group with AI" button).

```
"Group the tabs"
"Organize the tabs"
"Groupe les onglets"
```

### Sessions

#### Save
```
"Save the session"
"Save session Work"
"Sauvegarde la session"
```

#### Restore
```
"Restore the session"
"Open the session"
"Load session Work"
```

### Workspaces

#### Create
```
"Create a Dev workspace"
"New workspace Design"
```

#### Open
```
"Open the Dev workspace"
"Load the Design workspace"
```

## Free conversation

If your message doesn't match any command, it is treated as a **free conversation** with the AI, which responds taking your current tabs into account.
