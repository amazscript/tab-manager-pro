# Sessions

Save and restore the complete state of your browser at any time.

## Save a session

1. Open the popup > **Sessions** tab
2. Give your session a name (e.g., "Client X Project")
3. Click **Save**

The session captures:
- All open **windows**
- All **tabs** (URL, title, pinned status)
- All **tab groups** (name, color)
- The number of windows and tabs

## Restore a session

Click the **Restore** button next to the desired session. The extension:

1. Opens a new window
2. Recreates all tabs
3. Rebuilds groups with their original colors
4. Restores pinned tabs

::: info
Restoration opens a new window without closing existing windows.
:::

## Manage sessions

- **Delete**: Click the delete button to remove a session
- Sessions are stored locally in your browser
- No limit on the number of sessions

## Use cases

- **End of day**: Save your work context before closing Chrome
- **Project switch**: Move from one context to another without losing your tabs
- **Shared machine**: Each user can restore their own environment
- **Before an update**: Save your tabs before restarting Chrome
