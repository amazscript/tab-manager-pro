# Workspaces

Workspaces let you define **thematic work environments** that you can open and update at will.

## Create a workspace

1. Open the popup > **Workspaces** tab
2. Name your workspace (e.g., "Project Alpha")
3. Choose a **color** from the 8 available
4. Click **Create**

The workspace captures the URLs of all tabs in the current window.

### Available colors

`blue` `red` `green` `yellow` `purple` `pink` `cyan` `orange`

## Open a workspace

Click **Open** next to a workspace. The extension:

1. Opens a **new window**
2. Loads all tabs from the workspace
3. Automatically groups them with the workspace name and color

## Update a workspace

Added or removed tabs from your project? Click **Update** to sync the workspace with your current tabs.

## Sessions vs Workspaces

| | Sessions | Workspaces |
|---|---|---|
| **Purpose** | Complete snapshot | URL collection per project |
| **Content** | Windows + tabs + groups | List of URLs |
| **Usage** | Save/restore a state | Return to a work context |
| **Grouping** | Preserves existing groups | Creates a group automatically |

::: tip When to use what?
- **Session**: "I want to get back to exactly this browser state"
- **Workspace**: "I always want to have these sites at hand for this project"
:::
