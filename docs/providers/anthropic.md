# Anthropic (Claude)

## Configuration

1. Create an account on [console.anthropic.com](https://console.anthropic.com/)
2. Generate an API key in **Settings > API Keys**
3. In the extension, select **Anthropic** and paste your key
4. Click **Save**

## Technical details

| Parameter | Value |
|---|---|
| Model | `claude-sonnet-4-20250514` |
| Endpoint | `https://api.anthropic.com/v1/messages` |
| Max tokens (grouping) | 1000 |
| Max tokens (chat) | 2000 |

## Pricing

Claude uses a credit system. See the [Anthropic pricing page](https://www.anthropic.com/pricing) for details. The cost per grouping request is generally less than $0.005.

## Advantages

- Excellent context understanding
- Highly relevant and well-named groupings
- Native high-quality multilingual support
