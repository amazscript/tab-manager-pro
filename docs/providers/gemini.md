# Google Gemini

## Configuration

1. Go to [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Generate a free API key
3. In the extension, select **Gemini** and paste your key
4. Click **Save**

## Technical details

| Parameter | Value |
|---|---|
| Model | `gemini-2.0-flash` |
| Endpoint | `https://generativelanguage.googleapis.com/v1beta/models/` |
| Authentication | API key as URL parameter |

## Pricing

Gemini offers a **generous free quota** for developers. It is the most cost-effective provider to get started.

## Advantages

- **Free** within the quota limit
- Very fast Flash model
- Good grouping quality
- Ideal for testing the extension with no financial commitment

::: tip Recommended for beginners
If you don't know which provider to choose, start with Gemini: it's free and the quality is very good.
:::
