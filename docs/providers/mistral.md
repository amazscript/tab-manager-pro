# Mistral AI

## Configuration

1. Create an account on [console.mistral.ai](https://console.mistral.ai/)
2. Generate an API key
3. In the extension, select **Mistral** and paste your key
4. Click **Save**

## Technical details

| Parameter | Value |
|---|---|
| Model | `mistral-small-latest` |
| Endpoint | `https://api.mistral.ai/v1/chat/completions` |
| Authentication | Bearer token |

## Pricing

See the [Mistral pricing page](https://mistral.ai/pricing/) for details. Mistral Small offers a good quality/price balance.

## Advantages

- French company, data hosted in Europe
- Fast and affordable Small model
- Good quality for French language
- OpenAI-compatible API
