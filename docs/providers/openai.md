# OpenAI (GPT)

## Configuration

1. Create an account on [platform.openai.com](https://platform.openai.com/)
2. Generate an API key in **API Keys**
3. In the extension, select **OpenAI** and paste your key
4. Click **Save**

## Technical details

| Parameter | Value |
|---|---|
| Model | `gpt-4o-mini` |
| Endpoint | `https://api.openai.com/v1/chat/completions` |
| Max tokens (grouping) | 1000 |
| Max tokens (chat) | 2000 |

## Pricing

GPT-4o-mini is one of OpenAI's most affordable models. See the [pricing page](https://openai.com/pricing) for details.

## Advantages

- Fast and cost-effective model
- Good grouping quality
- Large community and documentation
- Compatible with OpenAI-compatible endpoints (configurable baseUrl)
